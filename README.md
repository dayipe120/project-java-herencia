
Descripción del proyecto

Este proyecto demuestra el uso de herencia en Java, modelando diferentes componentes de un vehículo (motor, transmisión, neumáticos, chasis y sistema de frenos).
Cada clase hereda de una clase base llamada ComponenteVehicular, y además implementa comportamientos específicos según su naturaleza.


src/
 ├── ComponenteVehicular.java
 ├── Motor.java
 ├── Transmision.java
 ├── Neumatico.java
 ├── Chasis.java
 ├── SistemaFrenos.java
 └── Main.java
